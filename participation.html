Participating in this Website

This website has just moved to this Github Repository. To participate in this project at this time you'll need a Github account from which you can
  • post issues
  • contact other Clue Dispensors
  • clone this repository
  • improve your clone of the repository
  • send pull requests to the official repository

Yes, git and github are pretty geeky and complicated, creating a barrier blocking some people from participating in this and other projects. Some of our friends are working on something easier called The Wicci System but it isn't done yet. In fact, it needs some more geeky folks to help! If you know of some, or perhaps you are one, please contact href="https://github.com/GregDavidson">the Wicci Project Coordinator.
